# Модуль Main

## Описание
Этот модуль содержит основные компоненты для работы main-процесса Electron. Он обеспечивает инициализацию приложения, создание главного окна, управление жизненным циклом приложения и обработку событий логирования.

## Структура

- **index.js** - Точка входа модуля, экспортирует все основные функции
- **core.js** - Основные функции инициализации приложения
- **createMainWindow.js** - Создание главного окна приложения
- **lifecycle.js** - Управление жизненным циклом приложения
- **logHandlers.js** - Обработчики IPC для системы логирования

## Использование

Модуль Main используется в mainHub.js - центральном хабе приложения. Для использования модуля достаточно импортировать необходимые функции из index.js:

```javascript
import { 
    initializeMainModule,
    setupLogInterception, 
    initializeApp, 
    registerLogHandlers 
} from "./Main/index.js";
```

## Функции

### initializeMainModule()
Инициализирует все необходимые компоненты Main-модуля.

### setupLogInterception()
Настраивает перехват логов для отслеживания ошибок и предупреждений.

### initializeApp(isDev)
Инициализирует основные компоненты приложения.
- `isDev` - Флаг режима разработки

### registerLogHandlers()
Регистрирует обработчики IPC для системы логирования.

### createMainWindow(isDev)
Создает главное окно приложения.
- `isDev` - Флаг режима разработки
- Возвращает: `BrowserWindow` - Объект главного окна

### setupAppLifecycle(isDev)
Настраивает обработчики событий жизненного цикла приложения.
- `isDev` - Флаг режима разработки
